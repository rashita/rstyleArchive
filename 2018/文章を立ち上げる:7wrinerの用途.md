今回は、文章の執筆法にいってみよう。

<h2>材料を集める</h2>

まず、一つのラインに書きたい文章を構成するメモを書き並べていく。サイズは多少小さい方がいいだろうが、文章みたいなものが入ってきても構わない。

<a href="https://rashita.net/blog/?attachment_id=24323" rel="attachment wp-att-24323"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-9.png" alt="" width="1244" height="739" class="alignnone size-full wp-image-24323" /></a>

続いてそれらをじっくり眺める。そして、近しいテーマだと思えるものを近づけていく。

<h2>カードを移動する</h2>

ある程度、数が集まり始めたら、テーマのうちの一つを横のラインに移動させる。大丈夫。順番みたいなものは、今は気にしないでいい。ただ塊をつくることだけを考えればいい。

<a href="https://rashita.net/blog/?attachment_id=24324" rel="attachment wp-att-24324"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-10.png" alt="" width="1256" height="392" class="alignnone size-full wp-image-24324" /></a>

しかしこのままだと、二つのラインにしか動かせない。別のラインに移動しようと思えば、いったん左右どちらかに移動させた後で表示をシフトし、さらに移動させることが必要だ。それはさすがに面倒である。

二つ方法がある。まずは、ラインそのものを移動させる方法だ。

command(winならalt) + ↓ を押すと、７つのライン全部が表示される。これはオーバービューモードと呼ばれている。

<a href="https://rashita.net/blog/?attachment_id=24325" rel="attachment wp-att-24325"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-11.png" alt="" width="1415" height="672" class="alignnone size-full wp-image-24325" /></a>

ここでは、ラインそのものの移動が可能となっている。

よって、ある程度入れ終わったラインを別の場所に移動させて、空白のラインを横に持ってくればいい。もう一度、command(winならalt) + ↓ で表示は通常に戻る。
※この戻り方でないとラインの移動情報が保存されないので注意。

<a href="https://rashita.net/blog/?attachment_id=24326" rel="attachment wp-att-24326"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-12.png" alt="" width="1333" height="684" class="alignnone size-full wp-image-24326" /></a>

あとはこれを繰り返していくだけだ。

それすらも面倒ならば、オーバービューモードで直接カードを移動させてしまう。

もう一度、もう一度、command(winならalt) + ↓ でオーバービューを表示させ、左上の「line mode」をクリックして、「card mode」に切り替える。こうすると、ラインの移動ができなくなる代わりに、カードの移動が可能となる。

<a href="https://rashita.net/blog/?attachment_id=24327" rel="attachment wp-att-24327"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-13.png" alt="" width="1417" height="580" class="alignnone size-full wp-image-24327" /></a>

配置をおえたら、通常モードに戻り、各ラインの見出しを付けておこう。KJ法的精神がここでは活きてくる。

<a href="https://rashita.net/blog/?attachment_id=24335" rel="attachment wp-att-24335"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-20.png" alt="" width="1420" height="514" class="alignnone size-full wp-image-24335" /></a>
<h2>文章を肉付けする</h2>

とりあえずこれで全体像みたいなものはぼんやりできた。あとは文章を書いていく。

7wrinerは、カードに複数行を含められるので、配置してあるカードのそのまま追記してもいいし、それを見出し扱いにして新しい文章用のカードを追加してもいい。とりあえずここでは、そのまま追記していこう。Editボタンを押したのち、カードに追記していく。

<a href="https://rashita.net/blog/?attachment_id=24328" rel="attachment wp-att-24328"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-14.png" alt="" width="593" height="270" class="alignnone size-full wp-image-24328" /></a>

軽く書き込んで調子が出てきたら、そのカードをダブルクリックしてみる。すると、フルスクリーンモードに突入する。

<a href="https://rashita.net/blog/?attachment_id=24337" rel="attachment wp-att-24337"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-21.png" alt="" width="1274" height="475" class="alignnone size-full wp-image-24337" /></a>

ここでぐんぐん書き込んでいく。とりあえず書ききれたら、左上の青い三角ボタンを押す。すると通常モードに復帰する。

<a href="https://rashita.net/blog/?attachment_id=24329" rel="attachment wp-att-24329"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-15.png" alt="" width="551" height="717" class="alignnone size-full wp-image-24329" /></a>

こうやってどんどん肉付けしていくと、一枚ごとのカードは肥大化し、一覧性は悪くなる。心配ない。オーバービューモードでは、カードの冒頭部分しか表示されない。一枚カードがどれだけ長くなろうとも、オーバービューでの俯瞰性は保たれている。また、この点が、どういう粒度のカードを作ればいいのかの指針にもなるだろう。

<a href="https://rashita.net/blog/?attachment_id=24339" rel="attachment wp-att-24339"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-23.png" alt="" width="226" height="388" class="alignnone size-full wp-image-24339" /></a>

<h2>オーバーフロー</h2>

と、こういった感じで文章の執筆を進めていくわけだが、7wrinerは７つのラインしかない。たいていの文章は７つ領域があればなんとかなると思うが、そうでない場合もあるだろう。もし８つ以上のラインが必要であれば、プレミアム版を……というのは嘘で、今のところ実装は考えていない（実装自体はすぐにできるのだがあえてそれはやっていない）。７つ、という制限が大切だと考えているからだ。

で、もし、７つ以上のラインが必要であれば、その際は、どこかのラインを「一時保管場所」に送っておくとよい。

各ラインの下に「send」というボタンがあり、

<a href="https://rashita.net/blog/?attachment_id=24330" rel="attachment wp-att-24330"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-16.png" alt="" width="739" height="320" class="alignnone size-full wp-image-24330" /></a>

これを押すと、ラインはすっかり消えてしまう。

<a href="https://rashita.net/blog/?attachment_id=24331" rel="attachment wp-att-24331"><img src="https://rashita.net/blog/wp-content/uploads/2018/04/screenshot-17.png" alt="" width="577" height="480" class="alignnone size-full wp-image-24331" /></a>

で、「send」の横にある「call」ボタンからそのラインを呼び出すことができる。

この辺でうまくラインをやりくりすれば、それなりに進めていけるだろう。

<h2>さいごに</h2>

7wrinerは、カードは複数行を許容する。それでも断片の操作に問題がないように、いくつかの機能が実装されている。

よって、上記のようにして執筆を進めていくこともできる。

ポイントは、この7wrinerで情報を扱っている間は、章ごとの順番について厳密に考えなくても良い点だ。

リストのように上から下に並んでいるわけではないし、横の配置も自由に移動できる。1,2,3,4と番号が振っていないので、並び方よりもまずそれぞれの中身について意識を向けることができる。もちろん、ある程度書き上げた内容を別のラインに移動させることもできるので、半固定な状態で執筆を進めていける。そういうツールだ。

というわけで今回は7wrinerを用いた文章の書き方を説明してみた。次回は、タスク管理について紹介する予定である。
