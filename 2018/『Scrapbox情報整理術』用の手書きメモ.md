<a href="https://rashita.net/blog/?attachment_id=25150" rel="attachment wp-att-25150"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25150" /></a>

「はじめに」についてとりあえず書き出してはみたものの、あまり進まず。初期段階ではよくあります。

<a href="https://rashita.net/blog/?attachment_id=25155" rel="attachment wp-att-25155"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-5.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25155" /></a>

こちらも同様。

<a href="https://rashita.net/blog/?attachment_id=25151" rel="attachment wp-att-25151"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-1.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25151" /></a>

全体の構成の整理です。目次案のベースになっています。

<a href="https://rashita.net/blog/?attachment_id=25152" rel="attachment wp-att-25152"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-2.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25152" /></a>

序章の話の流れを整理しています。要素群は頭の中にあるのですが、それをどう並べていくのかの検討です。

<a href="https://rashita.net/blog/?attachment_id=25153" rel="attachment wp-att-25153"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-3.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25153" /></a>

同じことをもう一回やっています。うまく流すために何度も検討します。

<a href="https://rashita.net/blog/?attachment_id=25154" rel="attachment wp-att-25154"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-4.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25154" /></a>

全体像の要素確認です。

<a href="https://rashita.net/blog/?attachment_id=25156" rel="attachment wp-att-25156"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-6.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25156" /></a>

終章の要素を確認しています。

<a href="https://rashita.net/blog/?attachment_id=25157" rel="attachment wp-att-25157"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-7.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25157" /></a>

こちらは本文中に使った図版用のイラスト（指示）。執筆中はScrapboxのDrawing機能を使ったのですが、さすがにそれではあまりにラフなのでしっかり描き直しです。

<a href="https://rashita.net/blog/?attachment_id=25158" rel="attachment wp-att-25158"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/efd626aac0d72942abd9b23cdf55bbdc-8.jpg" alt="" width="1600" height="1200" class="alignnone size-full wp-image-25158" /></a>

こちらも同様。

ちなみに、Drawingで描いたものがこちら。

<a href="https://rashita.net/blog/?attachment_id=25161" rel="attachment wp-att-25161"><img src="https://rashita.net/blog/wp-content/uploads/2018/07/screenshot-9.png" alt="" width="715" height="388" class="alignnone size-full wp-image-25161" /></a>
