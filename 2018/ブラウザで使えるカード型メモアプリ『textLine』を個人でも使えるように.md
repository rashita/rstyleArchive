<a href="https://rashita.net/blog/?p=24212" title="textLine:デジタルカードツールとそのメソッド – R-style">以前</a>紹介した「textLine」のデモは、あくまでデモであって、データは共有されていました。もちろん、個人の用途では使えません。

というわけで、個人でも使える簡易バージョンを作ってみました。

<a href="http://honkure.net/tool/textLineDemo/index.html" title="textLine">textLine</a>

データはブラウザごとに保存されますので、ご自由にお使い頂けます。

<h2>簡単な操作の紹介</h2>

<a href="https://rashita.net/blog/?attachment_id=24250" rel="attachment wp-att-24250"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-31.png" alt="" width="1417" height="755" class="alignnone size-full wp-image-24250" /></a>

まずは「＋」ボタンをクリック。

<a href="https://rashita.net/blog/?attachment_id=24251" rel="attachment wp-att-24251"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-32.png" alt="" width="1425" height="756" class="alignnone size-full wp-image-24251" /></a>

入力フィールドが作成されるので、そこに記述。

<a href="https://rashita.net/blog/?attachment_id=24252" rel="attachment wp-att-24252"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-33.png" alt="" width="892" height="390" class="alignnone size-full wp-image-24252" /></a>

Macならcommand + return、WinならAlt + returnで内容確定。

<a href="https://rashita.net/blog/?attachment_id=24253" rel="attachment wp-att-24253"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-34.png" alt="" width="839" height="407" class="alignnone size-full wp-image-24253" /></a>

カードが作成されました。

<a href="https://rashita.net/blog/?attachment_id=24254" rel="attachment wp-att-24254"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-35.png" alt="" width="860" height="534" class="alignnone size-full wp-image-24254" /></a>

同じ要領でどんどんカードを作成してけます。

<a href="https://rashita.net/blog/?attachment_id=24255" rel="attachment wp-att-24255"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-36.png" alt="" width="839" height="582" class="alignnone size-full wp-image-24255" /></a>

カードはドラッグで並び替え可能です。

<a href="https://rashita.net/blog/?attachment_id=24256" rel="attachment wp-att-24256"><img src="https://rashita.net/blog/wp-content/uploads/2018/03/screenshot-37.png" alt="" width="918" height="280" class="alignnone size-full wp-image-24256" /></a>

一度作成したカードも、クリックすれば再編集可能です。フォーカスを移動したときに内容が確定されます。

以上。あと、うっすらと左上に×ボタンがついているので、それをクリックすればカードを削除できます。

<h2>さいごに</h2>

今私が使っているバージョンは、これよりもう少し機能的バージョンアップが図られているのですが、基本的な部分──「カードを作成して、操作する」──は同様ですので、メモツールとしてそこそこ使えるかと思います。

無論、不満もいっぱいあるかと思いますので、その辺についてはこの記事のコメント欄に書いていただくか、Twitterでリプライ投げていただければ、改善の参考にさせていただきます。

では、良きメモ生活を。
