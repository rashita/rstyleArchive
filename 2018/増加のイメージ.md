<h2>積み上げる</h2>

<a href="https://rashita.net/blog/?attachment_id=26299" rel="attachment wp-att-26299"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-2-700x518.png" alt="" width="700" height="518" class="alignnone size-large wp-image-26299" /></a>

少しずつブロックを積み上げていく形。階段状になっていれば、自分も上がっていけるし、降りていける。

<a href="https://rashita.net/blog/?attachment_id=26300" rel="attachment wp-att-26300"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-3-700x528.png" alt="" width="700" height="528" class="alignnone size-large wp-image-26300" /></a>

直線に積み上げられたものは、自分では上がれない。眺めるだけ。仮に何らかの手段でてっぺんに立っても、怖くて降りられないので、その白色のブロックタワーに籠もることになる。

<a href="https://rashita.net/blog/?attachment_id=26301" rel="attachment wp-att-26301"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-4-700x533.png" alt="" width="700" height="533" class="alignnone size-large wp-image-26301" /></a>

実際、階段状になっているためには、見えない部分が積まれていないといけない。でないと、どこかで崩れる。

<h2>前進する</h2>

<a href="https://rashita.net/blog/?attachment_id=26302" rel="attachment wp-att-26302"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-5-700x532.png" alt="" width="700" height="532" class="alignnone size-large wp-image-26302" /></a>

ある方向を向いて進んでいくイメージ。自分の位置が変わり、風景が変わる。

<a href="https://rashita.net/blog/?attachment_id=26303" rel="attachment wp-att-26303"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-6-700x533.png" alt="" width="700" height="533" class="alignnone size-large wp-image-26303" /></a>

速度が速いイメージ。風景が変わる速度も速くなる。速度が上がれば、やがて風景は目に入らなくなる。

<a href="https://rashita.net/blog/?attachment_id=26304" rel="attachment wp-att-26304"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-7-700x521.png" alt="" width="700" height="521" class="alignnone size-large wp-image-26304" /></a>

前ではなく、上にというイメージもある。すでにそこにある「山」に登っていく、という感覚。

<h2>掘り下げる</h2>

<a href="https://rashita.net/blog/?attachment_id=26305" rel="attachment wp-att-26305"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-8-700x531.png" alt="" width="700" height="531" class="alignnone size-large wp-image-26305" /></a>

地中深く潜り込んでいくイメージ。視野が広がったり、自分が高みにあがることはない。ただ、何が埋まっているのかは掘ってみるまではわからない。あと、あまりに深い穴を掘ると、（登りたくても）自分では登れなくなる。

<h2>広がる</h2>

<a href="https://rashita.net/blog/?attachment_id=26306" rel="attachment wp-att-26306"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-9-700x531.png" alt="" width="700" height="531" class="alignnone size-large wp-image-26306" /></a>

ある領域が拡大するイメージ。

<a href="https://rashita.net/blog/?attachment_id=26307" rel="attachment wp-att-26307"><img src="https://rashita.net/blog/wp-content/uploads/2018/12/screenshot-10-700x533.png" alt="" width="700" height="533" class="alignnone size-large wp-image-26307" /></a>

アレンジとして、ネットワークが広がったり、接続が密になったりするイメージもある。

■

さて、あなたにとっての「成長」のイメージはどれが相当するだろうか。それとも、他のイメージだろうか。
