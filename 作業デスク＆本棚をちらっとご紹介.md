作業用場所の公開記事ですね。書斎の公開といえば、『センセイの書斎』という本がありますが、お偉いセンセイに限らず、他の人の書斎をのぞき見るのは楽しいものです。

と、自分が楽しんでいるばかりでもなんなので、マクロスの「俺の書斎を見ろ！」的なノリで私も書いてみることにしましょう。

<h3>見取り図</h3>
まずは。全体の見取り図から。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/screenshot3.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/screenshot3-293x300.png" alt="" title="screenshot" width="293" height="300" class="alignnone size-medium wp-image-8334" /></a>

手書きで申し訳ありません。

だいたい配置はこんな感じ。正面に机（自作）がドンとあって、右手にサイドテープル、左手に本棚が並んでいます。

私が家で作業するときは常にこのスペースです。できれば、イスから一歩も立ち上がりたくありませんので、必要そうな本はなるべく手元に置くようにしております。

では、それぞれを簡単にご紹介

<h3>作業用デスク</h3>
作業机です。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194050.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194050.jpg" alt="" title="20120628194050" width="480" height="360" class="alignnone size-full wp-image-8336" /></a>

手前が作業ゾーン。ほぼ日手帳カズン、ノート、MacBookAir13が鎮座しています。

上の段が物置場所。

左側から行ってみましょう。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194128.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194128.jpg" alt="" title="20120628194128" width="360" height="480" class="alignnone size-full wp-image-8340" /></a>

三段トレーですね。最上段が紙のインボックスになっています。ここに入ってきたものは、スキャナにかけられ、保存する必要のあるもの以外は、ゴミ箱に送られます。

二段目はクリアフォルダ入れ。一番下が新品の用紙入れになっています。

その右はノート類入れ。アクティブなものだけがここに保管されます。
※後ろに強い影が出ているのは、デスクライトの影響です。

さて、ずずずいと右に視点を移動しましょう。中央部分になります。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194139.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194139.jpg" alt="" title="20120628194139" width="480" height="360" class="alignnone size-full wp-image-8342" /></a>
※写真がぼやけているのは仕様です。

iPadエトセトラの説明は不要でしょう。もう少し視点を右に。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194201.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194201.jpg" alt="" title="20120628194201" width="480" height="360" class="alignnone size-full wp-image-8344" /></a>

「ブレスト用七つ道具ボックス」とペン立てが並んでいます。七つ道具については、また別の機会に詳しくご紹介しましょう。VAIOのお下がりディスプレイはMBAの外部ディスプレイ用です。

ここからもう少し右に視点を動かすとサイドテープルに移動します。

<h3>サイドテーブル</h3>
<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194152.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194152.jpg" alt="" title="20120628194152" width="360" height="480" class="alignnone size-full wp-image-8343" /></a>

正面のデスクも、デカイ板とボックスの組み合わせで出来ていますが、サイドテーブルも同様です。作った時期が違うので、天板の色合いが全然違いますが、ご愛敬。安い板を後ろに立てかけて、コルクボード的な使い方をしています。

一応、お手軽な小物置き場所として使っているのですが、気が緩むと、すぐこんな感じになります。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194229.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194229.jpg" alt="" title="20120628194229" width="480" height="360" class="alignnone size-full wp-image-8345" /></a>

この辺は、時間をかけて徐々に消化していくことになります。そろそろ本棚に置き場所が無くなってきた、という裏事情もあるのですが。

<h3>本棚</h3>
本棚について詳しくやると、また長くなるので、簡単に写真だけ。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194315.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194315.jpg" alt="" title="20120628194315" width="360" height="480" class="alignnone size-full wp-image-8349" /></a>
※前後二段で本が詰まっています。

<a href="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194330.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/06/20120628194330.jpg" alt="" title="20120628194330" width="360" height="480" class="alignnone size-full wp-image-8350" /></a>
※奥の方は小説類が多い

ちなみに、この本棚の上にもボックスを置いて本が並べてあります。ちなみに、ラノベは専用の・・・まあ、それはいいですね。
<h3>さいごに</h3>
というわけで、作業机と本棚を紹介してみました。

他の人のも見てみたいですね。
