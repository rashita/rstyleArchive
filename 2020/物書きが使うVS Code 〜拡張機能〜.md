今回は拡張機能を紹介します。これまた深い沼が広がっている領域です。

<h2>拡張機能のインストール</h2>

拡張機能は、サイドバーにある小規模なテトリスのようなアイコンから導入できます。

<a href="https://rashita.net/blog/?attachment_id=30358" rel="attachment wp-att-30358"><img src="https://rashita.net/blog/wp-content/uploads/2020/09/screenshot-12.png" alt="" width="82" height="428" class="alignnone size-full wp-image-30358" /></a>

検索ボックスに文字を打ち込んでいけば順次絞り込まれていきますので、それっぽい言葉を何か打ち込んで探すのがよいでしょう（ex.git,memo,todo,……)。

<a href="https://rashita.net/blog/?attachment_id=30359" rel="attachment wp-att-30359"><img src="https://rashita.net/blog/wp-content/uploads/2020/09/screenshot-13.png" alt="" width="440" height="1574" class="alignnone size-full wp-image-30359" /></a>

また、何かしら動作を行おうとしたときに、「ねえねえ、それを扱う機能は標準ではついてないよ、拡張機能いっとく？」とVS Codeから提案されることもあります。その場合、だいたい素直にインストールしたほうが吉です。

導入した拡張機能をアンインストールしたい場合も、この画面からいけます。一時的に機能停止、あるいはWorkspaceのみで機能停止、といったことも可能です。

<a href="https://rashita.net/blog/?attachment_id=30356" rel="attachment wp-att-30356"><img src="https://rashita.net/blog/wp-content/uploads/2020/09/screenshot-11.png" alt="" width="370" height="196" class="alignnone size-full wp-image-30356" /></a>

<h2>My list</h2>

で、これらの拡張機能はユーザー有志によって作られており、ほんともう驚くくらいあります。細かいニッチなものもたくさんあります。用途によって、必要な機能も違うでしょう。

なので、私のリストだけさらしておきます。

<a href="https://rashita.net/blog/?attachment_id=30354" rel="attachment wp-att-30354"><img src="https://rashita.net/blog/wp-content/uploads/2020/09/screenshot-9.png" alt="" width="428" height="1294" class="alignnone size-full wp-image-30354" /></a>

いくつか紹介だけしておきましょう。

<a href="https://github.com/nobuhito/vscode.actionlock">nobuhito/vscode.actionlock: ActionLock is a magical command inspired from Howm.</a>

True、でリターンしたらFalseになる、的なやつです。

<a href="https://github.com/mjbvz/vscode-github-markdown-preview">mjbvz/vscode-github-markdown-preview: Changes VS Code's built-in markdown preview to match Github markdown rendering in style and content</a>

マークダウンプレビューをGitHubに寄せてくれる拡張です。「Markdown Checkboxes」が欲しくて導入しました。

<a href="https://github.com/mechatroner/vscode_rainbow_csv">mechatroner/vscode_rainbow_csv: &#x1f308;CSV - VS Code extension: Highlight CSV and TSV spreadsheet files in different rainbow colors</a>

劇的にCSVファイルが（人間的に）読みやすくなります。

<a href="https://rashita.net/blog/?attachment_id=30355" rel="attachment wp-att-30355"><img src="https://rashita.net/blog/wp-content/uploads/2020/09/screenshot-10-700x124.png" alt="" width="700" height="124" class="alignnone size-large wp-image-30355" /></a>

<a href="https://github.com/atsushieno/vscode-language-review">atsushieno/vscode-language-review: Re:VIEW language Support for Visual Studio Code. / issue/PRは日本語でも対応できます</a>

最近使い始めた「Re:VIEW」用の拡張機能です。

<a href="https://github.com/satokaz/vscode-memo-life-for-you">satokaz/vscode-memo-life-for-you: This extension is for executing some commands of memo (Memo Life For You) from VS Code.</a>

一日ごとのファイルを作り、そこにささっとメモできたり、検索できたりする拡張機能です。
