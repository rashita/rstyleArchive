おはようございます。DIYの時間です。

情報カードボックスってあこがれますよね。下の動画とかを見ていても、くぅ〜、こういう箱欲しいぜぇ〜、みたいな気持ちになってきます。

<iframe width="560" height="315" src="https://www.youtube.com/embed/cMnor_tTUzE" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

でも、案外いいサイズの箱って見当たらないですよね。だったら、作っちゃえ、ということで我らがダイソーです。

<a href="https://rashita.net/blog/?attachment_id=29049" rel="attachment wp-att-29049"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/701C129B-29B8-4207-8C47-BF236A6C73C6-700x525.jpg" alt="" width="700" height="525" class="alignnone size-large wp-image-29049" /></a>

「シューズBOX（〜25cm）」。100円均一なのに300円もしやがる高級品です。メタルスライムとかパーラメントとかブルーマウンテンとかシャンパンとか、その系統です（たぶん違う）。

で、このシューズBOX。同じくダイソーで売られている5×3サイズの情報カードが、どどんと横に二列並びます。

<a href="https://rashita.net/blog/?attachment_id=29050" rel="attachment wp-att-29050"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/A476FDD3-1EB2-45A0-8FEC-944B13E243C6-700x525.jpg" alt="" width="700" height="525" class="alignnone size-large wp-image-29050" /></a>

そうそう、これだよ、これ。このボリューム感。1000枚くらいでもへっちゃに保存できそうです。これで麗しき知の貯蔵庫を手に出きる……と思いきや、やっぱり靴入れなんで若干深さが深い。マリアナ海溝とまではいかなくても、若干カードの操作に不便そうです。

そこで召喚。

<a href="https://rashita.net/blog/?attachment_id=29051" rel="attachment wp-att-29051"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/F6BB4CD7-8E24-44C8-822D-31F1D6C77821-700x933.jpg" alt="" width="700" height="933" class="alignnone size-large wp-image-29051" /></a>

これまたダイソーで売っている「発泡なんでも板A3」。名前から感じられる万能感がハンパないこいつを使って、底上げします。背を高く見せるための靴をイメージしてみるとわかりやすいかもしれません。

というわけで、ジャン。

<a href="https://rashita.net/blog/?attachment_id=29052" rel="attachment wp-att-29052"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/27793C38-8B65-4451-9836-950C437BC563-700x525.jpg" alt="" width="700" height="525" class="alignnone size-large wp-image-29052" /></a>

左隅が欠けているのは、サイズ感を試すためにちょこちょこはめていたらうっかり力を入れすぎて割れてしまったのでは決してなく、こうして端っこがちぎれていると後から取り出しやすいからです。いいですか。割れてしまったあとに、「まあ、これはこれで便利だからいいか」と気がついたわけではありませんよ。

ふ〜。

でも、こうして一枚かませても、まだちょっと高さが足りていません。なので、カッターで切り落とした「発泡なんでも板A3」の残りをさらに下に敷き詰めて二段構えとします。

<a href="https://rashita.net/blog/?attachment_id=29053" rel="attachment wp-att-29053"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/EB4DACDE-4CC1-4005-A10D-0ACEA138638B-700x525.jpg" alt="" width="700" height="525" class="alignnone size-large wp-image-29053" /></a>

Oh……デリシャス！　ん、デリシャスは違うか。まあいいや。

これで、がっつりカードを置いておけるようになりました。が、さすがに仕切りがなにもないと不安ですね。そこで、ジャジャン。

<a href="https://rashita.net/blog/?attachment_id=29054" rel="attachment wp-att-29054"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/2F7CB18E-5C5A-476E-9AD3-9FF7139E50F3-700x933.jpg" alt="" width="700" height="933" class="alignnone size-large wp-image-29054" /></a>

これまたこれまたダイソーで売ってる「切って使える仕切り板３」です。なんかもー、ダイソーから一歩も動いてねーな。俺。まあ、ダイソーの中ではさんざん探し待ったわけだけども。

ともかく、情報カードのサイズが、125mm×75mmなので、高さ8cmの仕切りはちょうど良いですね。こいつをえいやとはめこみます。

<a href="https://rashita.net/blog/?attachment_id=29055" rel="attachment wp-att-29055"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/2A91DD92-8D3B-439E-B8D1-C5B41A81F7EF-700x525.jpg" alt="" width="700" height="525" class="alignnone size-large wp-image-29055" /></a>

おぉ、それっぽい。仕切り版がもう少し余っているので、さらに分節化しておきます。

<a href="https://rashita.net/blog/?attachment_id=29056" rel="attachment wp-att-29056"><img src="https://rashita.net/blog/wp-content/uploads/2019/06/1FFA8DE0-0A16-49C2-9ABE-9553C48D4499-700x525.jpg" alt="" width="700" height="525" class="alignnone size-large wp-image-29056" /></a>

で、できた。できたぞー。

というわけで、カードボックス自体は500円（税別）で完成しました。使った工具は、カッターとハサミだけ。ちょーかんたんですね。

あとは、これを机の上に置いておき、がつんがつん情報カードを放り込んでいくだけ。まあ、そっちの方が難しかったりするかもしれませんが。
