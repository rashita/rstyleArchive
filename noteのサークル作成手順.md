さっき終わらせたので備忘録がわりに。

■

まず、右上のメニューから「サークル」を選択。

<a href="https://rashita.net/blog/?attachment_id=29913" rel="attachment wp-att-29913"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot.png" alt="" width="277" height="517" class="alignnone size-full wp-image-29913" /></a>

「サークルをはじめる」をクリック。

<a href="https://rashita.net/blog/?attachment_id=29914" rel="attachment wp-att-29914"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot-1.png" alt="" width="699" height="656" class="alignnone size-large wp-image-29914" /></a>

「サークル名」と「説明」を入力。サークル名は、目的をぐっと絞ったものにするか、「倉下忠憲のサークル」のように汎用的なものにするかの選択がありそう。「説明」は、サークルに興味を持ってくれた人にきちんと情報提供する気持ちで。

<a href="https://rashita.net/blog/?attachment_id=29915" rel="attachment wp-att-29915"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot-2-700x725.png" alt="" width="700" height="725" class="alignnone size-large wp-image-29915" /></a>

私はまあ、こんな感じ。
＊「<a href="https://mm.hyuki.net/circle">結城浩のサークル</a>」のパクリです。

<a href="https://rashita.net/blog/?attachment_id=29916" rel="attachment wp-att-29916"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot-3.png" alt="" width="672" height="703" class="alignnone size-large wp-image-29916" /></a>

続いてプラン作成。最大三つまで作成できるらしいが、とりあえず一つを。

<a href="https://rashita.net/blog/?attachment_id=29917" rel="attachment wp-att-29917"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot-4.png" alt="" width="675" height="767" class="alignnone size-large wp-image-29917" /></a>

まず、画像。「会員証」とあるが、プランのアイコンであり、複数プランを作るなら何かしら差異感をもたらすのが良さそう。星の数が増えるとか、色がリッチになるとか、何かそういう感じ。サイズは、1280×720あたりが良いのではないか。

プラン名もシンプルにして、説明でそれを補助する。複数プランを予定しているなら、〜〜の方はこちら、みたいな感じがよいのではないか。

<a href="https://rashita.net/blog/?attachment_id=29918" rel="attachment wp-att-29918"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot-5.png" alt="" width="696" height="566" class="alignnone size-large wp-image-29918" /></a>

でもって、月額の会費を決める。正直、100円以上なら、あんまり変わらない気もする。で、プラン内容だが、現状noteのサークル機能では差をつけにくいので、この辺は考え所。とりあえず、掲示板が使える、というのが最低限の特典だろう。あとは、公開非公開を選択し、お礼のメッセージを入力しておく。ここでは、ファーストな行動の指針を示しておくと良いだろう。「ぜひ、自己紹介を書き込んでください」というのもあるし、「そういうのは気にしなくていいですよ」というのもある。とりあえず、参加者は最初戸惑うものなので、その辺のフォローは大切だろう。

<a href="https://rashita.net/blog/?attachment_id=29919" rel="attachment wp-att-29919"><img src="https://rashita.net/blog/wp-content/uploads/2020/03/screenshot-6.png" alt="" width="694" height="655" class="alignnone size-large wp-image-29919" /></a>

あとは、審査のための情報をいくつか入力してそれで終わりである。

■

というわけで、私は今審査待ちの状況なので、いつまでたってもサークル始めないな〜と思われたら、審査に落ちたんだなと思ってください。では、では。
