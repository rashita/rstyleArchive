自分が普段使っているノートを公開するという企画があるようです。おもしろそうですね。・・・・・・

よろしい、ならば公開だ
征くぞ　諸君

<h3>どっさり</h3>
「普段使い」のノートだと、こんな感じです。

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100050.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100050.jpg" alt="" title="20120821100050" width="360" height="480" class="alignnone size-full wp-image-8723" /></a>

その他、買ってはみたものの・・・的なノートはたくさんありますが、そこまで足を伸ばす必要はないでしょう。

では、このノート群をいくつかカテゴリーに分けて紹介していきます。
<h3>手帳</h3>
<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100209.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100209.jpg" alt="" title="20120821100209" width="360" height="480" class="alignnone size-full wp-image-8725" /></a>

説明不要の、ほぼ日手帳2012。お気に入りのカバー付きです。行動記録などの「ログ」をメインに書き込んでいます。

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100203.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100203.jpg" alt="" title="20120821100203" width="480" height="360" class="alignnone size-full wp-image-8726" /></a>
<h3>手元のノート</h3>
手元によく置いてあるノートは３つあります。

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100228.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100228.jpg" alt="" title="20120821100228" width="480" height="360" class="alignnone size-full wp-image-8727" /></a>

まず、ノートカバー＋A5キャンパスノート＋リーガルパッド＋A4コピー用紙のセット。机の上にはだいたいこれが広がっています。
※<a href="https://rashita.net/blog/?p=8558">無印のノートカバーを改造して、３つのツールを持ち運べるように</a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100755.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100755.jpg" alt="" title="20120821100755" width="480" height="360" class="alignnone size-full wp-image-8728" /></a>
※A５キャンパスノートは、かなりラフな使い方

そして、普段持ち歩いているメモ的ノート。
※<a href="https://rashita.net/blog/?p=8622">おや、見たことのあるハードカバーのノートが</a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100413.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100413.jpg" alt="" title="20120821100413" width="360" height="480" class="alignnone size-full wp-image-8729" /></a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100739.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100739.jpg" alt="" title="20120821100739" width="480" height="360" class="alignnone size-full wp-image-8730" /></a>
※じゃじゃっと思考を書き出す。

最後の一つがNUboard。ホワイトボードノートです。ちゃちゃっと書いて、iPhoneのカメラで撮影→Evernoteの流れがGood。
※<a href="https://rashita.net/blog/?p=8642">NUboardゲット</a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100405.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100405.jpg" alt="" title="20120821100405" width="360" height="480" class="alignnone size-full wp-image-8731" /></a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120811224702.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120811224702.jpg" alt="" title="20120811224702" width="360" height="480" class="alignnone size-full wp-image-8732" /></a>

<h3>テーマノート</h3>
キャンバスノートB５サイズ群。

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100128.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100128.jpg" alt="" title="20120821100128" width="360" height="480" class="alignnone size-full wp-image-8733" /></a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100823.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100823.jpg" alt="" title="20120821100823" width="480" height="360" class="alignnone size-full wp-image-8734" /></a>

特定のテーマを勉強するときに使っています。学びのためのノートですね。最近はここに記入するようなことをメルマガの連載で書いているので、使用頻度は落ち込み気味。

<h3>育てるノート</h3>
<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100255.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100255.jpg" alt="" title="20120821100255" width="480" height="360" class="alignnone size-full wp-image-8735" /></a>

モレスキン・ラージについては、シゴタノ！に書く予定なので割愛。

モレスキン・ヴォランは、ある方からの頂き物なんですが（ありがとうございます）、「物書きの心得」みたいなものを集めています。本を読んだりして、「これは、自分にインストールしたい」というものを手書きで写している、ということですね。

<a href="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100349.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/08/20120821100349.jpg" alt="" title="20120821100349" width="480" height="360" class="alignnone size-full wp-image-8736" /></a>
<h3>さいごに</h3>
こうしてみると、馬鹿みたいに使っていますね。いいんです、ノートが好きだから。それが私の「選択」です。

ちなみにiPadの手書きノートアプリもあるんですが、今回は省略しておきました。

さてさて、みなさんは普段どんなノートを、どんな感じに使っておられるのでしょうか。
