魔が差す。

日常生活でよく見られる風景です。きっと誰を責めることもできません。だって、魔が差したんだから。

つまり、kobo Touchを購入した、ということです。

<h3>箱とご対面</h3>
今回は実用性のまったくないレビューを書く、という不退転の決意があるので、それをそのまま実行します。

ちょっとだけ役立ちそうなファーストインプレッション記事は日曜日のシゴタノ！をお待ちください。画面の写真などは、いつものようにFacebookページにアルバムとしてまとめてアップします。
※<a href="https://www.facebook.com/media/set/?set=a.421788651205394.109882.100366706680925&type=1">Facebookページの写真アルバム</a>

購入したのは「kobo Touch」のブルー。もう少し深い青が良かったのですが贅沢は言ってられません。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720120208.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720120208.jpg" alt="" title="20120720120208" width="360" height="480" class="alignnone size-full wp-image-8521" /></a>

箱の中身はシンプルですね。USBケーブルと紙は数枚だけ。いいですね。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720120414.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720120414.jpg" alt="" title="20120720120414" width="480" height="360" class="alignnone size-full wp-image-8522" /></a>

アクティベーションについてはまるっとすっ飛ばして、基本的な画面。

なるほど、なるほど。確かに文字は読みやすいですね。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720131921.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720131921.jpg" alt="" title="20120720131921" width="360" height=480" class="alignnone size-full wp-image-8523" /></a>

では、早速新しい書籍を購入しちゃいましょう。

Macのkoboアプリ。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot3.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot3.png" alt="" title="screenshot" width="500" height="340" class="alignnone size-full wp-image-8524" /></a>

いろいろラインナップはあるようです。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.1.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.1.png" alt="" title="screenshot.1" width="500" height="340" class="alignnone size-full wp-image-8525" /></a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.2.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.2.png" alt="" title="screenshot.2" width="500" height="340" class="alignnone size-full wp-image-8526" /></a>

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.3.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.3.png" alt="" title="screenshot.3" width="500" height="340" class="alignnone size-full wp-image-8527" /></a>

ドラッカーの赤本なんて全部集めれば本棚が相当圧迫されるので（されるだけの価値はありますが）、電子書籍でそろえるというのもアリなのかもしれません。他にもいろいろな本がならんでいます。

ただ、当たり前ですがリアルな「書店」のラインナップからすればかなり物足りない感じです。カテゴリー分けも「ん？」と思わないではない部分が多少あります。

まあ、この辺はスタートした段階なので絶対的な判断は保留したいところ。

とりあえず、何か一冊購入してみましょう。

で、今回選んだのが、この一冊。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.4.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.4.png" alt="" title="screenshot.4" width="500" height="340" class="alignnone size-full wp-image-8528" /></a>

・・・。えっと、・・・そう、たまたま。たまたま目に付いたんです。別にこのカバーを書店で買うのは躊躇するなぁ〜とか、そういう他意があったわけではありません。単なるたまたま。

クレジットカードを登録していると、操作はするする進みます。この辺はiPhoneなどでアプリを購入するのと同じ感覚ですね。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.5.png"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/screenshot.5.png" alt="" title="screenshot.5" width="500" height="340" class="alignnone size-full wp-image-8529" /></a>


UBSつなげっぱなしにしていたので、そのまま同期されていました。

で、これ。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720131617.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720131617.jpg" alt="" title="20120720131617" width="360" height="480" class="alignnone size-full wp-image-8534" /></a>

本文はこんな感じ。うむうむ、なかなか良いです。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720125532.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720125532.jpg" alt="" title="20120720125532" width="360" height="480" class="alignnone size-full wp-image-8535" /></a>

「さてさて、記事でも書こうか」と、スリープモードにすると、この画面が。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720125544.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720125544.jpg" alt="" title="20120720125544" width="360" height="480" class="alignnone size-full wp-image-8536" /></a>

おぉう。表紙が画面表示になっちゃうのか・・・。いろいろ設定を探してみると、一応オンオフの切り替えが可能とのこと。で、オフにしてみる。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720125637.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720125637.jpg" alt="" title="20120720125637" width="360" height="480" class="alignnone size-full wp-image-8537" /></a>

・・・。あまりにも貧相。この二択は新しく「究極の選択」に加えてもよいのではないだろうかという気がしないでもないです。まあ、本の表紙を選択しましたけどね。

機能はまだまだたくさんあって、まだこれから触るところですが、表示のフォントあたりは触っておきたいところ。

<a href="https://rashita.net/blog/wp-content/uploads/2012/07/20120720132021.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/07/20120720132021.jpg" alt="" title="20120720132021" width="360" height="480" class="alignnone size-full wp-image-8538" /></a>
※右下のボタンから変更可能です。

とりあえず「モリサワ リュウミン」というフォントを選択しておきました。「モリサワ ゴシック MB101」はちょっと文字の肉厚が強いので圧迫感があります。この辺は＿＿当たり前だけど＿＿好みの問題です。

<h3>さいごに</h3>
とりあえず、楽天ポイントのおかげで恐ろしく安く買えたので、人柱でも全然OKな感じです。少なくとも端末を触った感じではそれほど「君からは未来の匂いがしない」というイメージはありません。ごく普通に使える印象。

もちろん、電子書籍リーダーとして見た場合、電子書籍ストアがいかに充実し、かつ利用しやすいのかが大きな問題になるでしょう。が、そこら辺はバージョンアップと共に改善されるはずです（あくまではず）。

しかし、まだ本命とも呼べる対抗馬がその全貌を明らかにしていないので、過剰な期待をかけるのは難しいかもしれません。そういうストアの話を抜けば、わりに使いやすい端末ではないかと、個人的には思う次第です。kindle持ってないから断言はできませんが。
