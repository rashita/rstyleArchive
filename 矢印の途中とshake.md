たとえば、一冊の手帳を思い浮かべてみる。

<a href="https://rashita.net/blog/?attachment_id=26828" rel="attachment wp-att-26828"><img src="https://rashita.net/blog/wp-content/uploads/2019/02/screenshot-19-700x529.png" alt="" width="700" height="529" class="alignnone size-large wp-image-26828" /></a>

1月1日のページがあり、12月31日のページがある。n月m日はこの間のどこかに存在する。そのとき、n月m日よりも前の日は、すでに起きたことであり、つまりは過去である。対して後の日は、これから起こることであり、つまりは未来である（あるいは予定である）。

タスクシュート的なものはどうだろうか。

<a href="https://rashita.net/blog/?attachment_id=26829" rel="attachment wp-att-26829"><img src="https://rashita.net/blog/wp-content/uploads/2019/02/screenshot-20-700x534.png" alt="" width="700" height="534" class="alignnone size-large wp-image-26829" /></a>

時間のスケールは違うが、それでも同じである。時間が経つたびにnn:mmはどんどん下に移動し、下部の領域を塗りつぶしていく。未だ訪れていないものを、すでに訪れたものへと。

ここで行われているのは一体何だろうか。どのような「変換」だろうか。

話を拡げるために、shakeを参照する。

以下の画像は、<a href="https://takpluspluslog.blog.so-net.ne.jp/2016-08-29">夏の知的生産&ブログ祭り：Word Piece >>by Tak.：So-netブログ</a>より拝借したものを、加工してある。

<a href="https://rashita.net/blog/?attachment_id=26830" rel="attachment wp-att-26830"><img src="https://rashita.net/blog/wp-content/uploads/2019/02/screenshot-21-700x527.png" alt="" width="700" height="527" class="alignnone size-large wp-image-26830" /></a>


shakeは、トップダウンとボトムアップの行ったり来たりである。その往復を繰り返すことで、トップダウンだけで作られたコンテンツや、ボトムアップだけで作られたコンテンツとは別のものへと至る。あるいは、別の至り方を獲得する。

さて、この作業では、どんなインジケーターが進んでいるのだろうか。

<a href="https://rashita.net/blog/?attachment_id=26831" rel="attachment wp-att-26831"><img src="https://rashita.net/blog/wp-content/uploads/2019/02/screenshot-22-700x530.png" alt="" width="700" height="530" class="alignnone size-large wp-image-26831" /></a>

shakeが辿り着くのは、両極の一端ではない。少なくとも、そういう言い方はできる。

トップダウンをスタートとするならば、「こう書きたい」や「こう書くべき」が極となり、そこを目指して進んでいくこととなる。ボトムアップをスタートするならば、「こう書ける」「こう書いた」が極となり、それを増やしていくことで、完成に至る。

この二つを行ったり来たりするとは、何を意味するのだろうか。長い矢印の、ちょうど真ん中に着地することだろうか。

私には、とてもそんな風には思えない。shakeの過程を経て感じるのは、そのような均衡状態ではなく、一つの何かをやり遂げた、というような達成である。インジケーターは最後まで進んだのだ。

もちろん、それがある種のバランスを維持していることは間違いない。バランスとは中庸であり、それは均衡でもある。その点は間違いない。ただしそれは、一般的にイメージされる「理想と現実の間」というような単純なものではない。

<a href="https://rashita.net/blog/?attachment_id=26832" rel="attachment wp-att-26832"><img src="https://rashita.net/blog/wp-content/uploads/2019/02/screenshot-23-700x523.png" alt="" width="700" height="523" class="alignnone size-large wp-image-26832" /></a>

では、それは何なのか。

私たちが一日の始めにタスクリストを作り、それを完遂できないにせよ、ある程度は達成できたという状態は、何を意味しているのだろうか。

考えたいのは、そういうことなのだ。

もちろん現時点で結論はないが、このイメージの罠については指摘できる。それは、「未来」（理想）が固定的であるかのように扱われている、という点だろう。そのイメージを改変することが、話を先に進めるきっかけになりそうだ。
