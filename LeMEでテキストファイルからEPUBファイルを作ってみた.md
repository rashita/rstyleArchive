簡単にEPUBファイルが作れるという「LeME」の噂を聞きつけました。

<a href="http://relog.xii.jp/leme/">LeME | Let's make ebooks!</a>

実際に触ってみたところ、非常にお手軽です。ちょっとやってみましょう。

<h2>準備</h2>

LeMEは、サーバーにファイルをアップロードするタイプではなく、アプリケーションをダウンロードして使用するタイプです。

<a href="http://relog.xii.jp/leme/about/download/">ダウンロード | LeME</a>

環境にあったインストーラーをチョイスしてDLのち実行。とりあえず作成するだけならこれだけでもうOKです。EpubCheck、KindleGenをセットアップする場合は、「<a href="http://relog.xii.jp/leme/about/setup/">セットアップ | LeME</a>」のページを参照してください。今回はまるっと割愛します。

<h2>作成</h2>

では、実際にやってみましょう。

<a href="https://rashita.net/blog/?attachment_id=22866" rel="attachment wp-att-22866"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-15-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22866" /></a>

まずは「新規作成」。

<a href="https://rashita.net/blog/?attachment_id=22867" rel="attachment wp-att-22867"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-16-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22867" /></a>

こちらが基本的な画面です。なにかテキストファイルを追加してみましょう。

<a href="https://rashita.net/blog/?attachment_id=22868" rel="attachment wp-att-22868"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-17-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22868" /></a>

「<a href="https://conv.denshochan.com/downloads">Downloads - でんでんコンバーター</a>」からテキストファイルを拝借しました。見出しの設定方法が違うので、中身に少し手を加えていますが、基本的には同じです。

<a href="https://rashita.net/blog/?attachment_id=22869" rel="attachment wp-att-22869"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-18-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22869" /></a>

右サイドバーから「目次ページを挿入」を選択。見出しを拾って自動的に目次を作ってくれるようです。

<a href="https://rashita.net/blog/?attachment_id=22870" rel="attachment wp-att-22870"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-19-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22870" /></a>

さらに画像ファイルも追加して、「表紙に指定」を選択しておきます。基本的にはこれだけです。下のタブから「ステップ２：本の情報」に移動します。

<a href="https://rashita.net/blog/?attachment_id=22871" rel="attachment wp-att-22871"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-20-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22871" /></a>

タイトルや著者名、出版社名などを入力。さらに下のタブで「ステップ３：作成」へと移動。

<a href="https://rashita.net/blog/?attachment_id=22872" rel="attachment wp-att-22872"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-21-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22872" /></a>

作成するファイルの保存場所＆ファイル名がないので、任意でそれを設定します。

<a href="https://rashita.net/blog/?attachment_id=22873" rel="attachment wp-att-22873"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-22-500x402.png" alt="" width="500" height="402" class="alignnone size-medium wp-image-22873" /></a>

今回は、デスクトップの上に「sample.epub」というファイルで出力することにしました。で、「作成開始」。

<a href="https://rashita.net/blog/?attachment_id=22874" rel="attachment wp-att-22874"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-23-500x391.png" alt="" width="500" height="391" class="alignnone size-medium wp-image-22874" /></a>

ババっと、忍者が姿を消すぐらいの時間で作成完了。

<a href="https://rashita.net/blog/?attachment_id=22875" rel="attachment wp-att-22875"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-24-500x245.png" alt="" width="500" height="245" class="alignnone size-medium wp-image-22875" /></a>

きちんとファイルが作成されています。

<a href="https://rashita.net/blog/?attachment_id=22876" rel="attachment wp-att-22876"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-25-500x353.png" alt="" width="500" height="353" class="alignnone size-medium wp-image-22876" /></a>

iBooksで中身を覗いてみると、表紙の設定もうまくいっていますし、目次も動作しています。

<a href="https://rashita.net/blog/?attachment_id=22877" rel="attachment wp-att-22877"><img src="https://rashita.net/blog/wp-content/uploads/2017/09/screenshot-26.png" alt="" width="462" height="711" class="alignnone size-medium wp-image-22877" /></a>

もちろん本分もきちんと縦書きにレイアウトされています。問題ありません。

<h2>さいごに</h2>

というわけで、メチャ簡単にテキストファイルからEPUBファイルが作成できました。多少テキストファイルの作り方にコツがいりますが、その辺は以下のページでフォローされているのでそれをご覧ください。

<a href="http://relog.xii.jp/leme/making-guide/point-text/">テキスト編集のポイント（小説） | LeME</a>

でもって、このLeMEは、wordファイルからもEPUBファイルが作成できちゃいます。アウトライン構造やスタイルを利用できるということですね。残念ながら私はwordファイルを使う環境にないので、その辺の利用感は実際に試して頂くしかありませんが、今回ご紹介したようにテキストファイルからでもかなり簡単に作成できます。EPUB作成の強力な味方と考えて間違いないでしょう。

ちなみにLeMEは無料で使えるのですが、ライセンスを購入すると、自動的に本の巻末に追加される画像を無くすことができます。本格的にこのツールで本作りを進めていく場合はライセンス購入を検討してみてもよいかと思います。
