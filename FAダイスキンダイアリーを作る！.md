以前紹介した100円ショップのダイソーで発売されている「2013年ダイアリー」。

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104750.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104750.jpg" alt="" title="20121103104750" width="360" height="480" class="alignnone size-full wp-image-9144" /></a>

モレスキンそっくりの外観なので、ダイスキン・ダイアリーなどと呼ばれております。B６サイズでハードカバーの表紙、それにしおり紐とゴムバンドも付いてきてなんと105円。通信販売会社も真っ青のコストパフォーマンスです。
※スペックは→「<a href="https://rashita.net/blog/?p=9038">ダイスキンダイアリーをゲット</a>」

最近は、このノートを愛用しております。でもって、普段使うツールは、どんどんとカスタマイズしていくのが、カスタマイザーの宿命（さだめ）です。

いろいろやっているうちに、フルアーマー・ダイスキンダイアリー（FADD)みたいになってきました。

<h3>表紙</h3>
まずは表紙。紙の帯が付いていたので、それを切ってドットライナーで貼り付けました。

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104834.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104834.jpg" alt="" title="20121103104834" width="360" height="480" class="alignnone size-full wp-image-9145" /></a>

さらにお馴染みのシールをペタり。この辺はカスタマイズ初級編ですね。さらなるアレンジとしてプロジェクトを一つ片付けるたびに星マークのシールを貼っていってもいいかもしれません。最終的には赤く塗装することもありでしょう。

<h3>付箋＆ミニノートレール</h3>
表紙をめくったところ。

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104857.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104857.jpg" alt="" title="20121103104857" width="480" height="360" class="alignnone size-full wp-image-9146" /></a>

付箋に加えて、ミニメモをセットする場所が設けられています。別のノートからゴムバンドをパージさせ、二つに切って、マスキングテープで貼り付けてあります。

正直なところ重量耐性はあまりよろくありませんが、ミニメモぐらいなら平気です。ちなみにこのミニメモはリヒトラブさんの「ツイストリングノート」です。リングノート＋ルーズリーフという感じのアレですね。中身の入れ替えが可能なので、ページ数を抑えて厚み的な邪魔にならないようにしています。
※このメモについては→「<a href="https://rashita.net/blog/?p=5839">ツイストリング・ノート（メモサイズ）をゲット：使い方夢想</a>」

これで、小さいメモが必要な時は、ビームサービルを引き抜くようにノートからメモを出すことが可能です。

<h3>ポケット＆ホワイトボード</h3>
続いて、最後のページ。

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104932.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121103104932.jpg" alt="" title="20121103104932" width="480" height="360" class="alignnone size-full wp-image-9147" /></a>

まず左側。ポケットが付けてあります。こちらもダイソーで購入したもの。本来横向きに付けて使うクリアポケットを縦に貼り付けてあります。中身は5x3の情報カード。

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121020123807.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121020123807.jpg" alt="" title="20121020123807" width="360" height="480" class="alignnone size-full wp-image-9150" /></a>

続いて右側。

本当に、大変申し訳ないのですが、NUboardを分解して、その1ページ分のホワイトボードを貼り付けてあります。マスキングテープで貼り付けてあるのは、ノートを使い切った時、新しいノートに移行させるためです。
※NUboardについては→「<a href="https://rashita.net/blog/?p=8642">NUboardゲット</a>」。新バージョンが出るらしいのでそれをゲットします。

こうして貼り付けておくと、

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121103105025.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121103105025.jpg" alt="" title="20121103105025" width="480" height="360" class="alignnone size-full wp-image-9148" /></a>

こんな感じで、デイリーページを拡張して使うことができます。付箋を貼ってもいいですし、ホワイトボードマーカーで記入することもできます。まるでシールドのように襲いかかってくる割り込み案件からタスクリストを守ってくれるわけです。心強いですね。

<h3>B６情報カード</h3>
あと、B6サイズの情報カード（というかミニルーズリーフ）も数枚挟み込んでいます。やはり大きいサイズの紙は必要ですね。

<a href="https://rashita.net/blog/wp-content/uploads/2012/11/20121103105115.jpg"><img src="https://rashita.net/blog/wp-content/uploads/2012/11/20121103105115.jpg" alt="" title="20121103105115" width="480" height="360" class="alignnone size-full wp-image-9149" /></a>

今のところマルマンさんの「LOOSE FEAF2」を使っていますが、他に何かないか模索中です。折りたたんでおいて、広げたらA4サイズになる、みたいなアイテムを入れておけばファンネル的な応用力を持たせることができるかなと思います。

<h3>さいごに</h3>
というわけで、今回はダイスキンダイアリーのオリジナル改造について紹介してみました。

とりあえず旬物の文房具なので、すでにがっつり買い貯めこんでおります。それでも一冊105円なので、サイフにとっても優しいのです。まるでジムのようです（文房具は事務用品ですが）。

・・・。

お後がよろしいのかわかりませんが、とりあえず今回はここまで。また改造の手が入ったらご紹介してみます。
