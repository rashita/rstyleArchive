Dynalistの操作方法・使い方を説明していきます。

今回はメイン画面における項目の操作を紹介します。

<h2>画面構成</h2>

<a href="https://rashita.net/blog/?attachment_id=29500" rel="attachment wp-att-29500"><img src="https://rashita.net/blog/wp-content/uploads/2019/09/screenshot-11-700x494.png" alt="" width="700" height="494" class="alignnone size-large wp-image-29500" /></a>

これがメイン画面です。一番上に表示されている、ややサイズが大きい文字がタイトルです。ページに含まれる項目全体の見出しのようなものと考えれば良いでしょう。そのタイトルの下に、内容となる項目が並んでいます。

新規項目の追加は簡単で、何かの項目でリターンキー（改行キー）を押せば、その下に新しい項目が追加されます。ばんばん書きだしていきましょう。

<h2>移動</h2>

各項目の先頭には黒い丸ポッチがつきます。ビュレット（バレット）などと呼ばれています。この黒丸をマウスでドラッグし、別の場所に持っていけば、項目を移動させることができます。

<a href="https://gyazo.com/7bb6e8227000c5715564c937e04c98b2"><img src="https://i.gyazo.com/7bb6e8227000c5715564c937e04c98b2.gif" alt="Image from Gyazo" width="944"/></a>

<h2>インデント</h2>

項目は、横に一つ深くすることもできます。深くしたい項目の場所で、Tabキーを押せば項目が深くなります。深くなった項目で、リターンキーを押し、新しい項目を作成すると、同じ深さで項目を作成できます。

<a href="https://gyazo.com/51411724abdba3331ea16663c68fec4f"><img src="https://i.gyazo.com/51411724abdba3331ea16663c68fec4f.gif" alt="Image from Gyazo" width="992"/></a>

<h2>隠す</h2>

項目を深くすると、その項目が一つ上の「子要素」になります。一種のグルーピングと考えてください。ひとつにまとまる、ということです。

そのようにまとめると、「子要素」を隠すことができます。一度見えなくなるようにしてしまう、ということです。もちろん、単に隠しているだけですから、ボタン一つで再表示させることも可能です。

<a href="https://gyazo.com/08e1336828644d44bd4f3ab1525c5703"><img src="https://i.gyazo.com/08e1336828644d44bd4f3ab1525c5703.gif" alt="Image from Gyazo" width="820"/></a>

<h2>グループ移動</h2>

さらに、一つにまとめたブループは、それ全体を一手間で移動させることもできます。

<a href="https://gyazo.com/e0bb2dc06d320a5f84b06bf8212fae54"><img src="https://i.gyazo.com/e0bb2dc06d320a5f84b06bf8212fae54.gif" alt="Image from Gyazo" width="996"/></a>

<h2>ズーム</h2>

Dynalsitでは、作成された各項目に「ズームイン」することができます。項目にマウスを合わせると左側に虫眼鏡のボタンが表示されるので、それをクリックするとズームインします。

<a href="https://gyazo.com/f602ef6ba3e6fd1051e6ff015cecad9b"><img src="https://i.gyazo.com/f602ef6ba3e6fd1051e6ff015cecad9b.gif" alt="Image from Gyazo" width="996"/></a>

ズームインすると、項目全体が表示されるのではなく、その項目とグループだけが表示されます。上部に表示されるテキストをクリックすれば、もともとの状態に戻ることも可能です。

<h2>さいご</h2>

まずは、項目に関する基本操作を確認してみました。

リターンキーで次々に項目を作成していき、それを移動したり、グルーピングしたり、グルーピングしたものを移動したりと、自由に操作していけます。

あまりに項目が多い場合は子要素を隠すこともできますし、また、特定の項目だけに絞って入力を進めたい場合などはズーム機能を使うこともできます。

Dynalistとはそのようなツールです。
