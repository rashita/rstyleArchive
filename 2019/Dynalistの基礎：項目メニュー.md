項目にマウスを合わせると、左側に二つのボタンが表示されます。二つのボタンの内、外側にあるものがメニューボタンです。

<a href="https://gyazo.com/40e95a1bb6785bda93dc9b6f1265f1e7"><img src="https://i.gyazo.com/40e95a1bb6785bda93dc9b6f1265f1e7.gif" alt="Image from Gyazo" width="1000"/></a>

ここから項目に関するさまざまな操作が行えます。

<h2>collapse/expand</h2>

collapseとexpandは下位項目に関する操作です。collapseは下位項目を閉じ、expandは下位項目を開きます。

collapse all/expand allは、下位項目に含まれる下位項目すべてを閉じる/開く操作です。

<a href="https://gyazo.com/5c4d4715b65f831a2ea024f21839267e"><img src="https://i.gyazo.com/5c4d4715b65f831a2ea024f21839267e.gif" alt="Image from Gyazo" width="1000"/></a>

<h2>Zoom in</h2>

その項目にズームします。メニューボタンの右に表示されているボタンと同じ動作です。

<h2>Add note</h2>

「note」を追加します。noteについては、別の回で説明します。

<h2>Delete</h2>

その項目を削除します。下位項目が含まれる場合はそれごと削除されます。

<h2>Delete checked items</h2>

チェックボックスがチェックされた項目を削除します。チェックボックスについては、別の回で説明します。

<h2>Add bookmarks</h2>

ブックマークに追加します。ブックマークについては、別の回で説明します。

<h2>sort</h2>

項目を一定の規則に従って並べ替えます。たとえば、アルファベット順や作成日順などで並べ替えることが可能です。

<h2>Move to</h2>

項目を別の場所に移動させます。

<a href="https://rashita.net/blog/?attachment_id=29525" rel="attachment wp-att-29525"><img src="https://rashita.net/blog/wp-content/uploads/2019/09/screenshot-20.png" alt="" width="635" height="342" class="alignnone size-full wp-image-29525" /></a>

この機能を使うことで、別のドキュメントに項目を移動させられます。

<h2>add checkbokx to children</h2>

子要素にチェックボックスを追加します。以下のようになります。

<a href="https://rashita.net/blog/?attachment_id=29526" rel="attachment wp-att-29526"><img src="https://rashita.net/blog/wp-content/uploads/2019/09/screenshot-21.png" alt="" width="556" height="182" class="alignnone size-full wp-image-29526" /></a>

<h2>check off</h2>

チェックボックスを、チェック済みにできます。

<a href="https://rashita.net/blog/?attachment_id=29527" rel="attachment wp-att-29527"><img src="https://rashita.net/blog/wp-content/uploads/2019/09/screenshot-22.png" alt="" width="465" height="169" class="alignnone size-full wp-image-29527" /></a>

マウスクリックでチェック済みにすることもできますし、項目上でcommand + return(Mac)と入力することでも同様の結果が得られます。

<h2>Number children</h2>

子要素を番号付きリストにします。たとえば、以下のようになります。

<a href="https://gyazo.com/1aeb10637c623314b4f91a889d0fff5c"><img src="https://i.gyazo.com/1aeb10637c623314b4f91a889d0fff5c.gif" alt="Image from Gyazo" width="876"/></a>

<h2>Get link</h2>

その項目のURLを取得できます。

<a href="https://rashita.net/blog/?attachment_id=29528" rel="attachment wp-att-29528"><img src="https://rashita.net/blog/wp-content/uploads/2019/09/screenshot-23-700x361.png" alt="" width="700" height="361" class="alignnone size-large wp-image-29528" /></a>

そのURLにアクセスすると、項目にズームした状態で表示されます。

<h2>Export</h2>

項目の内容を外部に出力するための機能です。詳しくは別の回で説明します。

<h2>Set as inbox</h2>

その項目を「inbox」に設定します。

inboxに指定された項目は、一種の「郵便受け」となり、たとえば何からの機能によって外部から項目が新規追加された場合に、この項目の子要素として追加されることになります。

<h2>カラーと見出し</h2>

一番下にあるのは、項目の色や見出しを代えるボタンです。いろいろ押して試してみるとよいでしょう。
