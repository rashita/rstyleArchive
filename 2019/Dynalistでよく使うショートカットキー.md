Dynalistを本格的に使い始めたので、まずショートカットキーを覚えました。WorkFlowyと違うものもあるので注意が必要です。


とりあえず、よく使っているのは以下。

<a href="https://rashita.net/blog/?attachment_id=28275" rel="attachment wp-att-28275"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-11-700x681.png" alt="" width="700" height="681" class="alignnone size-large wp-image-28275" /></a>

ちなみにMacです。

<h3>項目の完了（チェックマークも）</h3>

<strong>command + return</strong>

<a href="https://rashita.net/blog/?attachment_id=28276" rel="attachment wp-att-28276"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-12.png" alt="" width="345" height="100" class="alignnone size-full wp-image-28276" /></a>

項目を完了扱いするショートカット。タスク管理的に使う場合は頻繁に登場します。

その項目にチェックボックスが付いているときも、同じショートカットで完了にできます。

<a href="https://rashita.net/blog/?attachment_id=28277" rel="attachment wp-att-28277"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-13.png" alt="" width="397" height="159" class="alignnone size-full wp-image-28277" /></a>

完了を未完了に戻す場合も、同じキーでちなみに、チェックボックスを挿入するのは、<strong>command + shift + c</strong> です。

<h3>項目の削除</h3>

<strong>command + shift + delete</strong>

カーソルがある行をまるっと削除します。

ちなみに、<strong>command + z</strong> で操作を一手戻せるので間違ったときなどは。

<h3>項目の上下移動</h3>

<strong>command + ↑↓</strong>

項目を上下に動かします。非常によく使いますね。

ちなみに、末端での操作がインデントになる場合があります。

たとえば以下の状態で、command + ↓ を押すと。

<a href="https://rashita.net/blog/?attachment_id=28278" rel="attachment wp-att-28278"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-14.png" alt="" width="362" height="147" class="alignnone size-full wp-image-28278" /></a>

このように項目が一つ上にあがります。

<a href="https://rashita.net/blog/?attachment_id=28279" rel="attachment wp-att-28279"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-15.png" alt="" width="372" height="203" class="alignnone size-large wp-image-28279" /></a>

で、さらに command + ↓ すると兄弟項目に移動します。

<a href="https://rashita.net/blog/?attachment_id=28280" rel="attachment wp-att-28280"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-16.png" alt="" width="378" height="197" class="alignnone size-full wp-image-28280" /></a>

<h3>項目の左右移動（インデント）</h3>

<strong>tab / shift + tab</strong>

項目の階層を上げ下げします。だいたいのアウトライナーがこの操作なので迷いはないでしょう。

<h3>下位項目の開閉</h3>

<strong>command + .</strong>

下位項目を開いたり、閉じたりします。

<a href="https://gyazo.com/e56381bff99d4b66fca959361d3728bb"><video alt="Video from Gyazo" width="582" autoplay muted loop playsinline controls><source src="https://i.gyazo.com/e56381bff99d4b66fca959361d3728bb.mp4" type="video/mp4" /></video></a>

<h3>項目にズームイン/ズームアウト</h3>

<strong>coomand + ] / command + [</strong>

その項目にズームイン、あるいはその項目からズームアウトします。

<a href="https://gyazo.com/b8f35486ea60bda8eb63ecc05d08b6d5"><video alt="Video from Gyazo" width="716" autoplay muted loop playsinline controls><source src="https://i.gyazo.com/b8f35486ea60bda8eb63ecc05d08b6d5.mp4" type="video/mp4" /></video></a>

<h3>他のドキュメントにジャンプ</h3>

<strong>coomand + o</strong>

<a href="https://gyazo.com/2f49990ffd319fa9fc254c97b7cbdf3e"><video alt="Video from Gyazo" width="660" autoplay muted loop playsinline controls><source src="https://i.gyazo.com/2f49990ffd319fa9fc254c97b7cbdf3e.mp4" type="video/mp4" /></video></a>

他のドキュメントにジャンプできます。動作を見る限り、ズーム状態は保持されるようです（5月2019年にジャンプしているが、5月13日（月） が表示されている）

この機能のおかげで、長いアウトラインをいちいちスクロールさせる必要がありませんし、サイドパネルを開く必要もありません。めちゃ便利です。

<h3>他の項目下位に項目を移動する</h3>

<strong>command + shift + m</strong>

<a href="https://rashita.net/blog/?attachment_id=28283" rel="attachment wp-att-28283"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-18.png" alt="" width="591" height="340" class="alignnone size-full wp-image-28283" /></a>

ショートカットを叩くと、こういう窓が表示されて、送り先の項目を検索して指定できます。これも長大なアウトラインを操作するには便利ですが、たぶん項目名の付け方にちょっとした工夫が必要かもしれません。

<h3>サイドパネルの開閉</h3>

<strong>command + shift + f</strong>

ジャンプ機能があるおかげであまり使わないのですが、上のショートカットで左のパネルが開閉します。パネルにドラッグ、あるいはパネルからドラッグしたいときに便利です。

<h3>項目にノートを追加</h3>

<strong>shift + return</strong>

項目の完了操作と間違えやすいので注意。以下のように、下位項目ではない付属情報を追加できます。

<a href="https://rashita.net/blog/?attachment_id=28281" rel="attachment wp-att-28281"><img src="https://rashita.net/blog/wp-content/uploads/2019/05/screenshot-17.png" alt="" width="448" height="145" class="alignnone size-full wp-image-28281" /></a>

<h2>まとめ</h2>

というわけで、実際に私がよく使っているショートカットだけをまとめてみました。全体は <strong>command + / </strong>で表示されますのでそちらで確認してください。

よくよくご覧になるとわかりますが、だいたい主要なことは command + なんとか でできるようになっています。それを理解しておくと覚えやすいかもしれません。
